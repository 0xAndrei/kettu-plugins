# Kettu Plugins

This repository hosts Kettu-compatible plugins over GitHub Pages.

## Plugins

- `hello-kettu`: a minimal example plugin

## Install URL

Install the example plugin from:

`https://0xAndrei.github.io/kettu-plugins/hello-kettu`

## Editing

Edit source files in `plugins/hello-kettu`, then rebuild with:

`npm install`

`npm run build`
